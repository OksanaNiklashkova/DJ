# DJ_Project

Этот проект предоставляет возможности для обработки данных электронной торговли

## Описание
В проекте представлено приложение, обеспечивающее работу интернет-магазина мини-приложений.
В верхней части страницы нажмите на кнопку "Регистрация" или авторизуйтесь на сайте.
При успешной регистрации на электронную почту вы получите приветственное письмо от сервиса.
Авторизованные пользователи могут просматривать предложенные на сайте продукты и создавать свои.
Пользователь, создавший продукт, автоматически получает статус его владельца и в любой момент может отредактировать его карточку: добавить или убрать реквизиты, изменить картинку и описание.

## Точка входа
Для перехода в веб-приложение запустите команду `python manage.py runserver`  и нажмите на ссылку в консоли `http://127.0.0.1:8000/`

## Установка

Для установки и запуска проекта необходимо выполнить следующие шаги:

1.  **Клонируйте репозиторий:**

    ```
    https://github.com/OksanaNiklashkova/DJ.git
    ```

2.  **Перейдите в папку проекта:**

    ```
    cd Project_DJ
    ```

3.  **Установите зависимости с помощью Poetry:**

    ```
    poetry install
    ```
4. **Запустите команду запуска сервера, чтобы перейти в веб-приложение**
    ```
    python manage.py runserver
    ```
